DB판정 데이터 탑재 상태

DB는 data/ 폴더의 CSV 파일에서 읽어옵니다. GitHub Pages에 배포하면 자동으로 불러옵니다.

DB를 불러오지 못했나요? 파일 직접 선택

로컬에서 file://로 열면 보안정책상 자동 로드가 막힐 수 있습니다. 그럴 땐 아래에서 CSV 파일들을 직접 선택하세요.

1모델명 조회

여러 모델은 쉼표·띄어쓰기·탭·줄바꿈 등 어떤 기호로 구분해도 각각 처리됩니다.

2판정정보 조회 결과

DB의 자가판정/전문판정 결과를 모델별로 확인합니다.

3판정 신청 (판정정보 없는 모델)

판정 이력이 없는 모델은 판정이 필요합니다. 모델(품목)별로 아래 항목을 입력하세요. 무게는 필수입니다.

📎각 모델의 모델명이 포함된 사양서를 반드시 함께 첨부하여 제출하세요.

4위험 점검 · 상황허가 의심징후

상황허가 의심징후 점검은 수출 1회당 1번 작성합니다(품목별 아님). 최종사용자 정보를 입력하세요.
