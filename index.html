수출 판정 · 사전신고 점검 도구

모델명을 입력하면 판정정보를 확인하고, 필요한 신청·신고 양식을 자동으로 채워드립니다. 마지막에 결과 보고서를 내려받아 담당자에게 제출하세요.

DB 탑재 상태

아래 DB는 data/ 폴더의 CSV 파일에서 읽어옵니다. GitHub Pages에 배포하면 자동으로 불러옵니다. (불러오지 못한 경우 파일을 직접 선택할 수 있습니다.)
DB를 불러오지 못했나요? 파일 직접 선택
로컬에서 file://로 열면 보안정책상 자동 로드가 막힐 수 있습니다. 그럴 땐 아래에서 CSV 파일들을 직접 선택하세요.

1모델명 입력

여러 모델명을 한 번에 입력할 수 있습니다. 쉼표, 띄어쓰기, 탭, 줄바꿈으로 구분하세요.

2판정정보 조회 결과

DB의 자가판정/전문판정 결과를 모델별로 확인합니다.

3판정 신청 (판정정보가 없는 모델)

판정 이력이 없는 모델은 판정이 필요합니다. 모델(품목)별로 아래 항목을 입력하세요. 무게는 필수입니다.
📎 각 모델의 모델명이 포함된 사양서를 반드시 함께 첨부하여 제출하세요.

4위험 점검 · 상황허가 의심징후

상황허가 의심징후 점검은 수출 1회당 1번 작성합니다(품목별 아님). 최종사용자 정보를 입력하세요.

최종사용자 정보

상황허가 의심징후 (12가지)

해당되는 항목에 체크하세요.

5사전신고

모델별 수출 내역

판정번호는 DB에서 자동 매칭됩니다(없으면 비어 있음). 수량·단가(USD)·용도를 입력하세요.
